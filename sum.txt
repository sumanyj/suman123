Echo command:

echo command is used to display message/content on the terminal
and by using echo command we can redirect the content to a file

syntax: echo <message/content>

ex: echo hello-everyone

Above commmand will display the message hello-everyone on the terminal

echo <content> > <file-name>

Above command will write the content to a file by replacing the actual content

echo <content> >> <file-name>

Above command will write the content to a file without replacing the actual content

Note: If a file is not present then it will create a file and move the content

Cat command

cat <file-name>

Above command will display the content of a file on the terminal

cat -n <file-name>

Above command will display the content of a file on the terminal along with line numbers

cat -b <file-name>

Above command will display the content of a file on the terminal along with line numbers
where the content is present


cat -s <file-name>

Above command will display the content of the file on a terminal by supressing multiple
empty lines to a single empty line

cat -s -n <file-name>

Above will display gthe content of a file on the terminal,where it will supress the multiple
empty lines and also it will assign the line numbers

cat <file1.txt> <file2.txt>

Above command will display the content together for  all the files mentioned in the command

More command:

Note: Press down arrow key to display the next page content on the terminal and we can use q for quit
